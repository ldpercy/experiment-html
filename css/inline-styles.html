inline-styles

How do inline styles apply to content?

		Inline style tags
		-----------------

		Not something you'd normally do, but I might possibly have a use-case for it (depending on how it works/until I find something better).
		Also, just curious.

		Basically if you have a bunch of mini-stylesheets in plain old `style` tags inline in the html - how do the rules affect the content around them?

		### Result

		Inline styles do **not** get applied progressively.

		They're combined and applied as a whole to the document.

		So you can't use something fairly dumb like this to target specific bits of the document, which I was speculatively aiming for.

A paragraph, no styles yet. I'll put a 'style' tag after it to change the colour.

Another paragraph; there's a 'style' tag just above this time, between this and the previous p tag. The 'p' style applies to both paragraphs.

And again, this time to monospace.

And finally override to green.

Rightyo then. There isn't anything like 'styles apply until until overridden', it doesn't work that way. The inline styles probably get combined into a single stylesheet, with it's own override cascade, then applied to the document as a whole. Makes sense, but haven't tried anything this basic in a very long time.
